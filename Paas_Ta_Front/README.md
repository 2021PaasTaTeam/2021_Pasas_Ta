# 2021 파스타 클라우드 공모전


### Front-End

### `cd paas_ta_front`

### `cd paas`

### `npm install`

### `npm start`

Open [http://localhost:3000](http://localhost:3000) 


---

### Back-End(for chat)

### `cd paas_ta_front`

### `cd paas`

### `cd server`

### `node server.js`

Open [http://localhost:3001](http://localhost:3001)
